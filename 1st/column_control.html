コラム 制御文

条件により，処理の流れを変えるのが制御文です． Ruby では，条件を「偽」以外は「真」として扱います． 「真」は true または「偽」以外です． 「偽」は，false または nil になります． C や Perl のように，数値の「0」については，「偽」になりませんので注意ください． もし 0 を偽として扱いたい場合には Numeric#nonzero? で判断してください．

比較など式を評価する場合には「真/偽」を返します． 条件の確認にはよく使用されることになります．

if

if 条件 [then]
  式...
[elsif 条件 [then]
  式...]
[else
  式...]
end

条件により処理を行うのが if です． elsif により複数の条件を確認することも可能です． 多くの条件を使用する場合は，case を使用すると読みやすくなります．

while

while 条件 [do]
  式...
end

条件が「真」の間繰り返されます．

break, next

ループ構造から抜けだすのが break です． ループを終了します． next は，ループ構造からループの先頭に戻ります．

case

case 式
when 式[, 式...] [then]
  式...
[else
  式...]
end

case で指定された式と，when で指定された式が一致することで実行されます．
