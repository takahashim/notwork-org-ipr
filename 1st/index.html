  • おわりに

HTTP

  • httpget.rb 指定した HTTP URL をHTTPで取得し、ヘッダも含めて表示する
  • httpget.c httpget.rb を C で実装したもの
  • toy_http_server.rb 簡単な HTTP サーバ

DAYTIME

  • daytime.rb DAYTIME クライアント
  • daytime_server.rb DAYTIME サーバ

ECHO

TCP

  • tcp_echo.rb ECHO クライアント
  • tcp_echo_server_repeat.rb ECHO サーバ (反復型)
  • tcp_echo_server_concurrent.rb ECHO サーバ (並行型)

UDP

  • udp_echo.rb ECHO クライアント (connectしない版)
  • udp_echo_by_connect.rb ECHO クライアント (connectする版)
  • udp_echo_server.rb ECHO サーバ (反復型のみ)

FTP

  • getrfc.rb 指定した番号のRFCを取得する
